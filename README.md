# KtClaw 🦞

Kotlin 实现的 OpenClaw - 个人 AI 助手平台

## 技术栈

| 组件 | 版本 |
|------|------|
| Kotlin | 2.3.10 |
| JDK | 25 |
| Ktor | 3.4.1 |
| Gradle | 9.4.0 |
| PostgreSQL | 16+ |
| Exposed | 0.58.0 |
| R2DBC | 1.0.0.RELEASE |

## 特性

- 🚀 基于 Ktor 3.4.1 的高性能 Web 服务
- 🗄️ PostgreSQL + Exposed + R2DBC 响应式数据库访问
- 🤖 多 Agent 协作架构
- 💬 QQ Bot 集成
- 🎨 Ktor HTML DSL 管理后台
- ⚙️ 多格式配置支持 (TOML/YAML/JSON)
- 🔌 SPI + 热加载扩展机制

## 快速开始

```bash
./gradlew run
```
